Get Lost

An app to find new ways to get lost in a familar place or while traveling. Uses html5 geolocation sevices, Google Maps Javascript Api v3.0, Google Routes Api and Google Maps Drawing Library to provide a randomly generated route and directions for the user to Get lost within.
